1. Review Inheritance

Inheritance adalah konsep pewarisan dimana suatu class dapat mewarisi atribut dan method dari class lain menggunakan keyword extends.

2. Pengertian Polimorfisme

Polimorfisme berarti banyak bentuk, yaitu kemampuan suatu objek untuk memiliki lebih dari satu bentuk perilaku.

3. Konsep Polimorfisme

Objek dari subclass dapat diperlakukan sebagai superclass sehingga memudahkan dalam penggunaan dan pengembangan program.

4. Jenis Polimorfisme

Terdapat dua jenis yaitu compile-time dan runtime, namun pada Dart yang digunakan adalah runtime polymorphism melalui overriding.

5. Polimorfisme dengan Inheritance

Method yang dipanggil akan menyesuaikan dengan objek yang digunakan walaupun menggunakan tipe parent.

6. Manfaat Polimorfisme

Polimorfisme membuat program lebih fleksibel, mudah dikembangkan, dan dapat menggunakan kembali kode.

7. Operator is dan as

Operator is digunakan untuk mengecek tipe objek, sedangkan as digunakan untuk mengubah tipe objek (casting).

8. Polimorfisme dengan List

List dapat digunakan untuk menyimpan berbagai objek dari subclass dalam satu tipe parent.

9. Pengertian Abstraksi

Abstraksi adalah proses menyembunyikan detail yang kompleks dan hanya menampilkan fungsi penting kepada pengguna.

10. Tujuan Abstraksi

Untuk menyederhanakan penggunaan program dan meningkatkan fleksibilitas dalam pengembangan.

11. Abstract Class

Abstract class adalah class yang tidak dapat dibuat objek secara langsung dan digunakan sebagai kerangka dasar.

12. Abstract Method

Method tanpa isi yang harus diimplementasikan oleh class turunan.

13. Interface

Interface digunakan untuk menentukan kemampuan yang harus dimiliki oleh suatu class.

14. Studi Kasus

Konsep ini digunakan dalam berbagai sistem seperti pembayaran digital dan perhitungan bentuk geometri.
