1. Review Inheritance

Inheritance adalah konsep pewarisan dimana class dapat mewarisi sifat dari class lain menggunakan keyword extends.

2. Pengertian Polimorfisme

Polimorfisme berarti banyak bentuk, dimana satu objek dapat memiliki perilaku yang berbeda.

3. Konsep Polimorfisme

Objek dari subclass dapat diperlakukan sebagai superclass sehingga lebih fleksibel.

4. Jenis Polimorfisme

Terdapat compile-time dan runtime, namun Dart menggunakan runtime polymorphism.

5. Polimorfisme dengan Inheritance

Method yang dipanggil tergantung objek aslinya meskipun menggunakan tipe parent.

6. Manfaat Polimorfisme

Mempermudah pengembangan, penggunaan ulang kode, dan membuat program lebih fleksibel.

7. Operator is dan as

Operator is digunakan untuk mengecek tipe objek, sedangkan as digunakan untuk casting tipe.

8. Polimorfisme dengan List

List dapat digunakan untuk menyimpan berbagai object dari subclass dalam satu tipe parent.

9. Pengertian Abstraksi

Abstraksi adalah menyembunyikan detail kompleks dan hanya menampilkan bagian penting.

10. Tujuan Abstraksi

Untuk mempermudah penggunaan program dan meningkatkan fleksibilitas.

11. Abstract Class

Abstract class adalah class yang tidak bisa dibuat objek langsung dan berisi method abstract.

12. Abstract Method

Method tanpa isi yang wajib diimplementasikan oleh subclass.

13. Interface

Interface digunakan untuk mendefinisikan kemampuan yang harus dimiliki class.

14. Studi Kasus

Penerapan konsep ini dapat digunakan pada sistem pembayaran dan bentuk geometri.
