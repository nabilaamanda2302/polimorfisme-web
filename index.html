1. Review Inheritance

Menggunakan extends untuk pewarisan, super untuk constructor, dan override untuk mengganti method.

2. Pengertian Polimorfisme

Polimorfisme berarti banyak bentuk, dimana satu objek memiliki banyak perilaku.

3. Konsep Polimorfisme

Objek subclass bisa diperlakukan sebagai superclass.

4. Jenis Polimorfisme

Compile-time dan runtime (yang digunakan di Dart adalah runtime).

5. Polimorfisme dengan Inheritance

Bentuk b = Lingkaran();
b.gambar();

6. Manfaat Polimorfisme

  • Fleksibilitas
  • Reusability
  • Maintainability

7. Operator is dan as

if (k is Mobil) {
  (k as Mobil).klakson();
}

8. Polimorfisme dengan List

Menggunakan List untuk menyimpan berbagai object subclass.

9. Pengertian Abstraksi

Menyembunyikan detail dan hanya menampilkan fungsi penting.

10. Tujuan Abstraksi

Fokus pada fungsi utama dan mempermudah penggunaan.

11. Abstract Class

abstract class Kendaraan {
  void start();
}

12. Abstract Method

Method tanpa isi yang wajib dioverride.

13. Interface

class Bird implements Flyable {
  void terbang() {
    print("Terbang");
  }
}

14. Studi Kasus

Penerapan pada sistem pembayaran dan bentuk geometri.

💖 Contoh Polimorfisme

class Hewan {

  }
}
